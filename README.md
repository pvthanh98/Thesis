## Class-based components
```javascript
class HelloReact extends React.components {
  constructor(props){
    super(props);
  }
  render() {
    return (
      <div>Hello React </div>
    );
  }
}
```

## Functional Components
```javascript
const HelloReact = (props) => {
  return (
    <div>Hello React </div>
  )
}


```
